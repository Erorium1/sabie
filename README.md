## 📖 О проекте

**Sabie** — это инновационное мобильное приложение, разработанное специально для будущих мам, которое позволяет отслеживать и записывать сердцебиение плода в домашних условиях с помощью портативного доплера. Приложение сочетает в себе медицинскую точность с интуитивно понятным интерфейсом, создавая уютную и безопасную среду для мониторинга здоровья малыша.

### 🎯 Цель проекта

Предоставить беременным женщинам удобный и надежный инструмент для:
- 📊 Мониторинга сердцебиения плода
- 📝 Ведения истории записей
- 📈 Отслеживания динамики развития
- 💡 Получения полезных рекомендаций
- 🤝 Обмена данными с лечащим врачом

## 🚀 Установка

### Требования

- Flutter SDK `^3.9.2` или выше
- Dart SDK `^3.9.2` или выше
- Android Studio / Xcode (для разработки)
- Android SDK / iOS SDK

### Шаги установки

1. **Клонируйте репозиторий**
   ```bash
   git clone <repository-url>
   cd sabie
   ```

2. **Установите зависимости**
   ```bash
   flutter pub get
   ```

3. **Проверьте подключение устройств**
   ```bash
   flutter devices
   ```

4. **Запустите приложение**
   ```bash
   flutter run
   ```

### Сборка для продакшена

**Android:**
```bash
flutter build apk --release
# или
flutter build appbundle --release
```

**iOS:**
```bash
flutter build ios --release
```
